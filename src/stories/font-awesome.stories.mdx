# Availible Icons

Icons are avaible via font-awesome. See the [FA react documentation](https://fontawesome.com/v5.15/how-to-use/on-the-web/using-with/react) for examples.

### Use

```jsx
import { FontAwesomeIcon as FAIcon } from '@fortawesome/react-fontawesome';
import { faCoffee } from '@fortawesome/free-solid-svg-icons';
// Use the Icon
<FAIcon icon={faCoffee} />;
```

### Result

> <FAIcon icon={faCoffee} />
